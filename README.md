# 소개
서울시에 등록된 온라인 쇼핑몰들에 대한 정보를 관리하는 서비스로 관리자는 csv 파일을 통해 편하게 데이터를 DB에 저장할 수 있다.<br>
업소 상태/ 전체 평가 필터링을 통하여 검색 가능하며 추가적으로 모니터링날짜 필터링을 통해 모니터링 주기가 지난 쇼핑몰들에 대한 관리가 가능하다.<br>
**도전기능**(시간부족으로 진행 못 함) : 캐싱(Caching)을 사용해서 자주 검색되는 쇼핑몰들에 대하여 캐시에 데이터를 저장해 응답 속도를 높이고 시스템 부하를 줄일 수 있다.<br>
